# GaiaNet Earth - NASA Digital Twin Dashboard (v2.0.0)

Welcome to **GaiaNet Earth**, a production-grade environmental intelligence platform. 

## Features
- **5-Tab Immersive Flow**: Earth, Insight, Timeline, Forecast, and Simulator.
- **Real-Time Data**: NASA FIRMS (Wildfires) and GIBS (Satellite) integration.
- **Universal Selection**: Click anywhere on the globe to analyze environmental health.
- **Simulation Engine**: Run "What-If" climate scenario models.

## Quick Start
1.  Ensure Python is installed.
2.  Double-click **`start_project.bat`** in the root folder.
3.  The dashboard will open automatically at `http://localhost:8080`.
